## 安全贡献指南

由于本仓库是公开访问的，为了保护隐私和信息安全，我们实施了严格的内容审查机制：

1. **提交前检查**：所有贡献者必须遵循 [安全审查清单](tools/security-review-checklist.md) 进行自查
2. **敏感信息过滤**：使用 `.gitignore` 自动排除常见的敏感文件类型
3. **自动化检测**：通过 Git Hooks 自动扫描潜在的敏感信息
4. **内容脱敏**：所有示例数据必须使用虚构信息替代真实数据

我们鼓励社区贡献，但也高度重视信息安全。请在提交 Pull Request 前仔细检查内容，确保不会泄露任何敏感信息。
